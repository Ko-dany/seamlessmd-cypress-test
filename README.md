This repository contains automated Cypress tests for the SeamlessMD patient login page.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Ko-dany/seamlessmd-cypress-test.git
cd seamlessmd-cypress-test
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run All Tests in Headless Mode

```bash
npx cypress run

npx cypress open # Open Interactive Mode
```
